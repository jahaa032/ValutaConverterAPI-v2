# ValutaConverterAPI-v2
Jeg valgte JavaScript, HTML og CSS fordi jeg kjente best til disse.

I denne applikasjonen kan brukeren konvertere mellom to valutaer. 

Brukeren velger valutaen man vil endre fra og til i to dropdown-menyene, og skriver beløpet i det første input-feltet.  
Det andre input-feltet viser resultatet av konverteringen og er deaktivert for redigering.  

Den har også en til knapp som konverterer pengene med bruk av ExchangeRate-API og viser hvor mye beløpet tilsvarer i den valgte valutaen.
